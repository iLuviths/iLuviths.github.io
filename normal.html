Distribución Normal (Campana de Gauss)

La distribución normal se utiliza cuando los datos se agrupan simétricamente alrededor de una media. Para calcular probabilidades necesitamos convertir el valor a la variable estandarizada Z.

Estandarización

Z = (X - μ) / σ

  • X: valor observado
  • μ: media
  • σ: desviación estándar
